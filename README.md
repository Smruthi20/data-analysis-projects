# data-analysis-projects

InSightCart Sales Analytics Dashboard
Advanced Excel | Power Query | Data Visualization

Project Overview

This project is an interactive InSightCart Sales Analytics Dashboard built using Excel and Power Query.
The dashboard helps visualize business performance by analyzing sales, profit trends, customer insights, and shipping patterns.
It is designed to support decision-making by highlighting key metrics that impact revenue and operational efficiency.

Objectives:

✔ Analyze sales performance across categories, sub-categories, and segments
✔ Identify top-performing products and high-value customers
✔ Compare sales performance across different regions and markets.
✔ Track year-over-year performance using slicers
✔ Provide a clean, interactive dashboard for quick business insights

Tools Used:

✔ Microsoft Excel
✔ Pivot Tables
✔ Pivot Charts
✔ Slicers
✔ Conditional Formatting
✔ Power Query (for data cleaning & transformation)
✔ Excel Functions (VLOOKUP, XLOOKUP, IF, DATE functions, etc.)

Key Insights:

✔ Identified top 6 revenue-generating sub-categories
✔ Analyzed customer purchase frequency to identify loyal customers
✔ Uncovered sales distribution across product categories
✔ Highlighted year-wise performance trends using slicers

How to Use the Dashboard:

✔ Download the Excel file
✔ Enable editing
✔ Use slicers to filter by:
  Year
  Category
  Region
  Customer Segment
✔ Interact with charts and visuals to explore insights

Key Features:

✔ Modern, clean UI with consistent color themes
✔ Multiple dynamic visuals (donut, bar, line charts)
✔ Automated data refresh using Power Query
✔ KPI-focused layout
✔ User-friendly for non-technical audiences
